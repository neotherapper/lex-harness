## 4. Download protocol (high level)

The full programmatic access protocol for each source lives in `09_ai_research/research/12_greek_legal_databases/{NN}_*.md`. This section is the **execution checklist** — read the linked research file before each step.

1. **Bootstrap directories:** `mkdir -p 05_legal_research/law_vault/{core,modules/{tenancy,consumer_protection,gdpr,tort_damages,corporate,tax_invoices,criminal_regulatory,procedure_eirinodikio},case_law/{ap,hdpa,cjeu,lower_courts}}`
2. **Core articles** (22 from §1) → scrape **kodiko.gr** by article URL. Protocol: see [`07_kodiko_gr.md`](research/12_greek_legal_databases/07_kodiko_gr.md). Verify each against lawspot.gr ([`06_lawspot.md`](research/12_greek_legal_databases/06_lawspot.md)).
3. **Module 1 (tenancy):** AK_574–624 via kodiko.gr (same protocol as step 2). Cross-check N_4172/2013 Art_24(7) reference inside AK_592 commentary.
4. **Module 2 (consumer):** Law 2251/1994 articles via kodiko.gr; N_5255/2025 via **search.et.gr** Legislation Search (year=2025, type=Νόμος, number=5255). Protocol: see [`01_search_et_gr.md`](research/12_greek_legal_databases/01_search_et_gr.md).
5. **Module 3 (GDPR):** `mcp__eur-lex__expert_search` for CELEX 32016R0679 → fetch via `resource://eurlex/document/32016R0679` in EL+EN. Then N_4624/2019 from e-nomothesia.gr (research file: gap, see §7 sub-list).
6. **Module 4 (tort):** AK_57, 937 from kodiko.gr. ΠΔ_212/2006 from search.et.gr Legislation Search (year=2006, type=Προεδρικό Διάταγμα, number=212).
7. **Module 5 (corporate):** N_4601/2019 Arts. 65 + 70 from search.et.gr (year=2019, type=Νόμος, number=4601). Verify on e-nomothesia.gr. Append YESTAY ΑΕ Note 10 admission text from `04_evidence/gemi_filings/YESTAY_PROSARTIMA_2024.pdf`.
8. **Module 6 (tax):** N_4172/2013 Art_24 from lawspot.gr; N_4308/2014 Arts 9+18 from taxheaven.gr; N_5177/2025 Arts 3+16 from search.et.gr; AADE Circular E_2094/2025 from aade.gr.
9. **Module 7 (criminal):** ΠΚ Arts 216, 362, 363, 367, 375, 386 from lawspot.gr Poinikos Kodikas section; verify kodiko.gr.
10. **Module 8 (procedure):** ΚΠολΔ articles from kodiko.gr. Law 5221/2025 from search.et.gr.
11. **ΑΠ inline decisions** (10 from §3): use **areiospagos.gr** decision search at `/nomologia/apofaseis.asp` (POST form: x_number, x_ETOS). Protocol: see [`08_areiospagos_gr.md`](research/12_greek_legal_databases/08_areiospagos_gr.md). Charset is windows-1253; results are HTML. Extract ratio + key quote; insert into module's `case_law_inline.md` (≤500 chars each). Fallback to dsanet.gr only if areiospagos.gr returns no result.
12. **CJEU decisions** (9 from §3): `mcp__eur-lex__expert_search` then `resource://eurlex/document/{CELEX}` for each ID; save EL+EN HTML to `case_law/cjeu/`.
13. **HDPA decisions** (5 from §3): `mcp__chrome-devtools__navigate_page` to dpa.gr SPA; `take_snapshot` + `evaluate_script` to extract decision PDF URLs; save to `case_law/hdpa/`. Backup: `site:dpa.gr filetype:pdf "[decision number]"` via WebSearch.
14. **Verification:** For every saved file run a checksum against the live source one week later via `mcp__fetch__fetch`; flag drift in `core/CHANGELOG.md`. **Citation rule:** primary source = ΦΕΚ via search.et.gr; verification source = kodiko.gr or lawspot.gr; never the other way around.
15. **ChromaDB ingestion:** All 36 cold-tier decisions + full ΑΚ + full ΚΠολΔ + full N_2251/1994 indexed via `scripts/build_vector_index.py` (extends existing email index).
